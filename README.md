# Express Base

This is a template de express. Use it with Docker or Docker Compose!


## Installation

```bash
npm start
```

or

```bash
npm run dev
```

## Installation with docker

```bash
docker build -t express-base .
docker run --name express-base -d -p 3000:3000 express-base
```

or

```bash
docker-compose up -d
```
